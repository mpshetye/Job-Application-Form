* Mongoose ^5.11.19v

* Nodemom ^2.0.7v
* MongoDB

	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm run dev
```
or

```
$ npm install
$ npm run start
```
You will then be able to access it at localhost
